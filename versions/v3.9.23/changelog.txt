# Prominence II: Hasturian Era v3.9.23 
**v3.9.23 - Performance & Changes**

This update adds a few smaller optimization tweaks as well as some new structures, incase you got bored. This Update also nerfs the matter cannon because it behaved a slight bit different to how we expected and fixes some smaller things in quests. The modpack should run easier on less ram and uses less resources when afking etc. 

**Warning! Computercraft will leave Prominence in the next 3 weeks! Next week will disable all recipes. Save all your Programs before its too late!**

New Structures! 
Many new Structures have been added, new villages, new ships and much more! Of course these structures only spawn in new chunks. Many of these structures also use mods like Create and Supplementaries, perfect to roach in!

12 New villages! Dozens of new structures in the Overworld, End and Nether! These structures, especially the Villages also use modded blocks and have a ton of refreshing and lively features - perfect for any Dungeon Roach!
